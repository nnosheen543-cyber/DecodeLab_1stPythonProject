# To-Do List Project

A simple command-line To-Do List application built using Python.  
This project allows users to add, view, and remove tasks using a menu-driven system.

## Features
1. Add new tasks
2. View all tasks
3. Remove tasks
4. Simple and beginner-friendly interface
5. Uses Python lists and loops

## Technologies Used 
Python

## File Structure
```text
todo.py
README.md
```
## How It Works
1. User selects an option from the menu.
2. Tasks are stored in a Python list.
3. Users can:
   ~Add tasks
   ~View tasks
   ~Remove tasks
4. The program keeps running until the user exits.

## Run the Project
Open terminal or command prompt and run:
```bash
python todo.py
```
